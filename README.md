# Web Project Homeland

🔗 **Live Project:**
https://israelliradev.github.io/web_project_homeland/

## 📖 Descrição

O **Web Project Homeland** é um projeto desenvolvido durante o sprint **Design Web Adaptativo**.

A página apresenta o tema **“Homeland”**, contando histórias de diferentes lugares e origens por meio de imagens e textos organizados em um layout moderno e limpo.

O objetivo do projeto foi recriar um layout de design e garantir que ele funcione corretamente em diferentes tamanhos de tela, aplicando técnicas de **redimensionamento de páginas** e **design responsivo**.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- **Princípios BEM Flat**
- Git
- GitHub Pages

## 📱 Design Web Adaptativo

O projeto foi desenvolvido utilizando **Responsive Web Design**, permitindo que o site se adapte a diferentes dispositivos:

- Desktop
- Tablet (até 1024px)
- Mobile (até 544px)
